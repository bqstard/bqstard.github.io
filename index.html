How to order 101

> Orders are done via DMs. If you're at this page, you should already know my Discord.
> If you've ordered art before (from anyone), you don't need to read all this.

What do I need for an order?

Checklist:
> Pose reference
> Hair + outfit references
> Face reference or description (optional if you're fine with generic features)
> Reference or description for the expression (if you want it to be different from the pose reference)
> Clear explanation of additional details. E.g. "Earring on the left side" - My left or the character's left?
> Background reference, if different from the pose reference (the background will be white / off-white if you don't specifically mention wanting a background)

Still not sure what to do?

> If you're unsure if I'm open or not, ask :)
> When you've gathered all the needed references, just let me know you'd like to order.
> Make sure you know what you want and have adequate references to show what you want. I'll only do small changes and adjustments after each stage of the drawing.
